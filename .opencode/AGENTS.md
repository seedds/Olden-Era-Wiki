# Olden Era Wiki — Flutter project

Unofficial companion wiki for Heroes of Might and Magic: Olden Era, built with Flutter.

## Project layout

```
assets/
  db/wiki.sqlite       # SQLite database (generated by extractor)
  images/              # Images extracted from game assets
    raw/sprite/
    raw/texture/
  icons.json           # Stat/faction icon key-to-path map (generated)
  app_icon.png
lib/                   # Dart source
scripts/
  extract_wiki_data.py # Data extractor (run from project root)
  take_screenshots.sh
.github/workflows/
  build.yml            # CI: builds Android APK and iOS IPA
```

## Data extraction

The extractor reads game files from the local Steam install and writes directly into `assets/`.

**Prerequisites:** Python 3, Pillow, UnityPy (`pip install Pillow UnityPy`)

**Run from the project root:**
```
python3 scripts/extract_wiki_data.py
```

Outputs written:
- `assets/db/wiki.sqlite`
- `assets/images/raw/sprite/` and `assets/images/raw/texture/`
- `assets/icons.json`

Intermediate work files land in `scripts/output/` (gitignored).

The `game_path` in `SETTINGS` at the top of the script points to the local CrossOver/Steam install — update it if the install location changes.

## Flutter

Minimum SDK: Dart ^3.5.0 / Flutter stable.

```
flutter pub get
flutter run
```

Assets are declared in `pubspec.yaml` under `flutter.assets`.
